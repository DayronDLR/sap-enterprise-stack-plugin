# Contributing

Thanks for your interest in the **SAP Enterprise Stack**! 🙌
*(Español abajo.)*

## Important: this repo is a generated distribution

This repository holds the **built artifact** of the plugin (commands, agents,
skills, hooks, MCP). It is **regenerated and published automatically** from a
separate development repository. So:

- Files here are **not edited by hand**.
- **Pull Requests to this repo are not merged** (they'd be overwritten on the
  next publish).

## How to contribute

- 🐛 **Bugs** → open an [issue](../../issues/new/choose) with the *Bug report*
  template.
- 💡 **Ideas / features / new agents or skills** → *Feature request* issue.
- ❓ **Questions** → an issue with the matching label.

For bugs, include: **plugin version** (`/plugin` shows it), OS, `pnpm --version`,
steps to reproduce, and what you expected.

## Environment

- `pnpm` on your PATH (MCP servers and hook linters use it).
- The quality hooks are **bash** scripts — on Windows you need Git Bash or WSL.
