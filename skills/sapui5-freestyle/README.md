## Skill Capabilities

- Scaffold completo de apps SAPUI5 FreeStyle (TypeScript o JavaScript)
- Patrones de UI: List/Detail, Dashboard KPIs, Formularios, Worklist
- Integración con OData V4 y CAP projects
- BaseController, routing, navegación
- i18n multi-locale y accesibilidad WCAG
- Validación con UI5 Linter y manifest validation
- Compliance con reglas enterprise del proyecto

- `fiori-mcp`: list_functionality, get_functionality_details, execute_functionality, search_docs
